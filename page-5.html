Install

To create your own zine, download the zip of this project here

Install Google Chrome web Server

Point Chrome Web Server to the folder of the project

You can then use a text editor like notepad, or the classic Sublime Text to edit each page of the zine as you would any other html file.

Upload the folder to any host (we really like neocities). Now everyone can look at your zine!

Print out your zine even side then odd side, staple in the center. You have yourself a zine!

Our goal is to make each PDF compatible with every Zine production services!
