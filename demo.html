A simple way to generate 8 page zines using HTML

Using Zine CSS

Download and include this CSS

This is what a template looks like:

<zine-wrapper>
  <zine-cover>
    <zine-left-page>
    </zine-left-page>
    <zine-right-page>
    </zine-right-page>
  </zine-cover>
  <zine-spread>
    <zine-left-page>
    </zine-left-page>
    <zine-right-page>
    </zine-right-page>
  </zine-spread>
  <zine-spread>
    <zine-left-page></zine-left-page>
    <zine-right-page></zine-right-page>
  </zine-spread>
  <zine-spread>
    <zine-left-page></zine-left-page>
    <zine-right-page></zine-right-page>
  </zine-spread>
</zine-wrapper>

Spread

To have an img or svg spread across the entire element just put the item in the zine-spread or zine-cover div
